# MangaBotForSlavyan

Telegram-бот для отслеживания манги с [ReManga](https://remanga.org/).

Что умеет:

- добавлять мангу по поиску с ReManga;
- хранить список манги для каждого пользователя отдельно;
- отправлять уведомления о новых главах;
- отмечать последнюю прочитанную главу для каждой манги;
- вручную выставлять прочитанную главу;
- показывать статус: что прочитано и сколько глав осталось.

## Команды

- `/start` - старт и главное меню
- `/help` - помощь
- `/search` - начать поиск манги
- `/list` - показать список отслеживаемой манги
- `/check` - вручную проверить новые главы
- `/cancel` - отменить текущий ввод

## Как работает уведомление

После добавления манги бот запоминает текущую последнюю известную главу и дальше уведомляет только о новых.

Прогресс чтения хранится отдельно у каждого пользователя.
